## 📌 About

ICRI (*Can I Run It*) is a desktop application built with **C#** and **WPF** designed to analyze system specifications and help determine whether a device can run specific software or games efficiently.

The project focuses on providing a clean user experience with fast system analysis and compatibility checking.

---

## ✨ Features

- 🔍 Detect system specifications
- ⚡ Fast compatibility analysis
- 🎮 Game/software requirement checking
- 🖥️ User-friendly desktop interface
- 📊 Performance overview
- 🚀 Lightweight application
